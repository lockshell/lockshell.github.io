V2Ray服务器端安装完成后，接下来就需要用V2Ray客户端与之相连了，下面我们介绍下V2Ray Mac客户端的下载、安装及使用。

安装之前，如果你还没有架设V2Ray服务器，那么可以参考以下链接文章，先在VPS服务器上安装V2Ray，本文介绍在此基础上展开：

  • 在AWS EC2上搭建V2ray+Caddy代理

V2Ray官方发布的Mac OS客户端，仅提供内核，没有图形化界面。 而V2RayX是一个集成V2Ray内核的图形化界面，安装在Mac OS之后，可以很方便的进行各项参数设置，快速连接V2Ray服务器。

V2RayX下载

V2RayX 是一个基于 V2Ray 内核的 Mac OS X 客户端。 用户可以通过界面生成配置文件，并且可以手动更新 V2Ray 内核。V2RayX 还可以配置系统代理。

V2RayX目前最新版本为1.5.1，版本随时更新，以实际为准。

V2RayX 1.5.1下载地址：

  • https://github.com/Cenmrev/V2RayX/releases/download/v1.5.1/V2RayX.app.zip

更多相关情况可以访问：

  • https://github.com/Cenmrev/V2RayX/releasesV

V2RayX安装和使用

  1. V2RayX下载完成后解压，得到V2RayX.app，将其拖动至程序目录进行安装。

因为软件不是来自于Mac自带的应用商店，打开程序时可能会有安全提示，如下：

需要前往“系统偏好设置”－“安全性和隐私”，点“仍要打开”，再点"install"安裝。

  1. 安装完成后打开V2RayX，桌面右上角会出现V2RayX图标。

  2. 点击桌面右上角的V2RayX图标，会弹出如下菜单，点击Configure进入设置

  3. 在设置界面，点击左下角的+号，添加服务器。 之后填入你 V2Ray 服务器的各项参数即可，包括：

  • Address (服务器域名和端口号): test.com 443
  • User ID：一个uuid字符串
  • alterId: 一般64
  • Security: 一般auto
  • Network（你的伪装类型）: 一般ws 或者 tcp

这些都是必填项跟你的 V2Ray 服务器一致。 最后点击 OK 保存你的 V2Ray 服务器配置，再次点击右上角通知栏的 V2RayX 的图标，点击Start V2Ray即可连接你的 V2Ray 服务器。

另外，菜单中的PAC Mode表示根据V2RayX内置规则，决定是否走代理，这种模式下，一般国内流量直连，部分国外网站走代理。 而Global Mode则表示所有流量走代理。可以根据自己的使用情况，选择其中一种模式。

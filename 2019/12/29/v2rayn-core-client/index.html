V2Ray服务器端安装完成后，接下来就需要用V2Ray客户端与之相连了，下面我们介绍下V2Ray Windows客户端的下载、安装及使用。

如果你还没有搭建好V2Ray服务器端，可以参考以下文章，本文内容在此基础上展开：

  • 在AWS EC2上搭建V2ray+Caddy代理

下载安装包

V2RayN 是一个基于 V2Ray 内核的 Windows 客户端。下载：Github

V2Ray内核 + V2RayN打包下载：

  • https://github.com/2dust/v2rayN/releases/download/3.12/v2rayN-Core.zip

安装客户端

  • 将下载的v2rayN-Core.zip压缩包解压至任意位置。
  • 运行V2RayN.exe即可。

提示： V2RayN的运行需要Microsoft .NET Framework 4.6或以上版本，如果运行时出现错误提示，很可能是你的系统中没有安装，先下载安装后再运行V2RayN：net48-offline-installer

新建Vmess服务

点击服务器菜单 -> 新建Vmess服务:

这里服务器配置基本跟下面截图基本一致；域名和用户id需要修改为v2ray服务端支持的用户ID和域名。

  • 地址: v.test.com (这里填写v2ray服务的域名)
  • 端口: 443
  • 用户ID: 36fdb836-xxxx-xxxx-xxxx-xxxxb71e737d
  • 额外ID: 64
  • 加密方式选: auto
  • 传输协议选: ws
  • 别名: v2-test (随便填写)
  • 路径: /ray
  • 传输层安全选: tls
  • 允许不安全连接选: true

参数设置

  1. 点击参数设置菜单 -> Core: 基础设置:

点击V2RayN菜单中的参数设置项，可以看到，默认情况下V2Ray的Socks代理已经开启，端口为1080。 本机中的其它软件，可以使用Socks协议，连接127.0.0.1:1080，然后就可以通过V2Ray代理上网了。

  • 本地端口: 1080
  • 其中自定义DNS: 8.8.8.8,8.8.4.4,localhost

示例图如下：

  1. 点击参数设置菜单 -> Core: 路由设置: 路由设置：可以在此选项下，设置V2Ray是否线绕过大陆地址及大陆IP。
  • 点击一键默认设置路由规则
  • 然后解析策略选择: IPOnDemand
  • 可在编辑框中根据需要添加路由规则

示例图如下：

  1. 对于普通用户，参数设置的另外几项，使用默认即可。
  • KCP设置：KCP加速相关，一般无需设置。
  • v2rayN设置：一般无需设置；可设置V2RayN开机自启，也可设置是否允许局域网连接，将代理共享给内网其它设备。

工作中遇到这样的子网ip和掩码：`192.168.20.0/24`，得在脑海中用二进制去算IP范围是多少，子网掩码是多少；有时还得在纸上画一画，耗时费力。
后来发现ipcalc这个神器，麻麻再也不用担心我算不动了。

一条命令，搞定网络类型、广播地址、子网IP范围等的计算，也很方便给小伙伴演示。

## 安装ipcalc

Mac OS
```
brew install ipcalc
```

Linux
```
sudo apt install ipcalc  
```

## 使用ipcalc
```
ipcalc 192.168.20.0/24
```
